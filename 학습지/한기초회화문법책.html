Grammar 02
      

〜이에요 / 예요

             
1. 의미 (意味)
      

명사 뒤에 붙어 「〜です」라는 뜻으로 사용되는 서술형 어미입니다. 주로 일상적인 대화에서 정중하게 자신이나 사물을 소개할 때 사용합니다.

      
2. 형태 규칙 (使い分け 규칙)
      

명사의 마지막 글자에 받침(末尾の子音)이 있는지 없는지에 따라 형태가 달라집니다.

             
          

💡 규칙 공식

          
                  
  • 받침 있음 (パッチムあり) + 이에요
    •               
  • 받침 없음 (パッチムなし) + 예요
    •           
      
      
3. 상세 설명 (解説)
      
              
  • 학생 (学-生 / 받침 있음): 마지막 글자 '생'에 'ㅇ' 받침이 있으므로 학생 + 이에요 = 학생이에요.이 됩니다. 연음 법칙에 의해 [학생이에요]로 부드럽게 발음됩니다.
    •           
  • 친구 (友-達 / 받침 없음): 마지막 글자 '구'에 받침이 없으므로 친구 + 예요 = 친구예요.가 됩니다.
    •       
      
4. 예문 (例文)
      
          

저는 한국 사람이에요.

          

私は韓国人입니다(です)。

      
      
          

이것은 제 휴대폰예요.

          

これは 제( 제 ) 携帯電話です。

      
      
          

여기가 우리 회사예요.

          

ここが私たちの会社입니다(です)。

2. 연습 문제 (練習問題)
      

[보기]를 참고하여 알맞은 형태로 문장을 완성하세요.

      

(보기: 학생 → 학생이에요 / 친구 → 친구예요)

      
          
Q1. 일본 사람 + ( 이에요 / 예요 )
          

→ 저는 일본 사람.

      
      
          
Q2. 의사 + ( 이에요 / 예요 )
          

→ 동생은 의사.

      
      
          
Q3. 선생님 + ( 이에요 / 예요 )
          

→ 그 분은 우리 선생님.

      
      
          
Q4. 커피 + ( 이에요 / 예요 )
          

→ 이것은 따뜻한 커피.
